# Editor visual

Os tipos de conector MDL 2.0 disponíveis são `link`, `always-link`,
`specialization`, `virtual-link` e `composition`.

Interface acessível sobre um command bus imutável. Operações sobre elementos, relações e grupos produzem estado semanticamente representável; referências inválidas são rejeitadas e remoções dependentes exigem confirmação explícita. Geometria não faz parte deste estado.
